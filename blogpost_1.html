Basics

To start off, lets get Hello World out of the way:

hello_world.py
print("Hello World!")
Program Output:
>>> Hello World!

Note: ">>>" is used to indicate the program output.

Well, that was easy. Note the lack of semicolons. Lets try a in if statement:

flow_control.py
if (True):
 print("True!")
else:
 print("False!")
Program Output:
>>>True!

No curly braces?
No semi-colons?
Capital "T" True?
What about System.out???

Alright, as Java programmers, there's a lot to digest here. Most importantly is the lack of curly-braces which guides so much of Java's structure. In Python, scope isn't determined by curly-braces, but by tabs. Notice how the places that you'd expect to find braces contain a single colon and then all of the text after it is tabbed up once? It looks completely different but most of the same concepts of scope that apply in Java apply here as well. The colon indicates a new block of code and deeper scope. All of the tabbed code that proceedes is part of that scope!

Not so crazy once it's explained right? Let's get a bit more involved, here's a weird take on the classic Fizz Buzz problem:

fizz_buzz.py
keywords = ["fizz", "buzz", "fizzbuzz"]
divisors = [3, 5]

for i in range(1, 16):
  answer = ""
  for divisor in divisors:
    if (i % divisor == 0):
      answer = answer + keywords[divisors.index(divisor)]
  if (answer == ""):
    print(i)
  else:
    print(answer)
Program Output:
>>> 1
>>> 2
>>> fizz
>>> 4
>>> buzz
>>> fizz
>>> 7

>>> fizz

>>> 11
>>> fizz

As you might start to see, the lack of curly-braces can make it seem difficult to read the code if you're not used to it. This code block begins to show a bit more of Python's syntax. We are introduced to arrays, a for loop, comparisons, and some built-in methods.

We also see that initializing variables in Python couldn't be easier, simply: variable = value no semicolons, and no types!

GUI

Yeah, yeah. If/Else, for loop, arrays, who cares? It's just boring text on a screen right? Let's get into the pretty parts of Python :)

sunrise.py
import turtle

turtle.setup(500, 500) # set the window size
turtle.speed("fastest")

turtle.shape("turtle")

turtle.up() # initial position of the turtle
turtle.right(90)
turtle.forward(500)
turtle.left(90)
turtle.forward(150)

for i in range(0, 50): # creating 50 circles to give a radial gradient effect
  rHexCode = hex(5 + (i * 4))[2:100] # slightly increment the red hex value on each iteration
  if (len(rHexCode) == 1):
    rHexCode = "0" + rHexCode
  gHexCode = "00"
  bHexCode = "00"
  turtle.fillcolor("#%s%s%s" % (rHexCode, gHexCode, bHexCode)) # combine our hex codes to set the color
  turtle.begin_fill() # draw the circle!
  turtle.circle(500 - (i * 8))
  turtle.end_fill()

turtle.exitonclick()

Program Output:
>>>

Well, I think it looks like a sunrise anyway. The previous code loops 50 times, creating circles that decrease in size and increase in redness on each iteration.
I had an issue on my first attempt at this where the whole screen would end up being the same color. This was because I was drawing the circles from smallest to largest. Each time a larger circle was drawn, it would cover the previous smaller circle! Logically, drawing the larger ones first solved that problem immediately.
Now that we're all familiar with the syntax, lets get into something a bit more fun. We all like geometry right?

shapes.py
import turtle

turtle.setup(500, 500)

background = turtle.Turtle()
background.speed(10)

background.up()
background.right(90)
background.forward(300)
background.left(90)

background.fillcolor("black")
background.begin_fill()
background.circle(500)
background.end_fill()

pentagon = turtle.Turtle()
pentagon.speed(10)

pentagon.up()
pentagon.forward(90)
pentagon.down()

pentagon.color("white")
pentagon.fillcolor("pink")

pentagon.begin_fill()
for side in range(0, 5):
  pentagon.forward(50)
  pentagon.right(72)
pentagon.end_fill()

pentagon.write('Pentagon')

square = turtle.Turtle()
square.speed(10)

square.up()
square.left(90)
square.forward(100)
square.down()

square.color("white")
square.fillcolor("red")

square.begin_fill()
for side in range(0, 4):
  square.forward(100)
  square.left(90)
square.end_fill()

square.write("Square")

star = turtle.Turtle()
star.speed(10)

star.up()
star.left(180)
star.forward(20)
star.left(90)
star.forward(100)
star.down()

angle = 100

star.color("white")
star.fillcolor("yellow")

star.begin_fill()
for side in range(0, 5):
  star.forward(80)
  star.right(angle)
  star.forward(80)
  star.right(72 - angle)
star.end_fill()

star.up()
star.settiltangle(0)
star.left(90)
star.forward(10)
star.down()

star.write("Star")
